Why Do You need to use this App

Indroduction

This App is designed to check the visual actulity of people eye. Snellen chart is the eye-chart which is used to perofme eye tests. Basically, App is designed to use people who are suffering from myopia. however, It could be used by anyone who needs to check the nearsightedness of the eyes.This App provides a facility for their users to store their test readings for further usage such as tracking, making hard copies, comparing, etc. Each of users' personal data and medical reports is protected by specific security rules and regulations. Every user must be register to the app by creating thier own personal account. After creating an user account once, who can access to the App and perfome thier acitivies such as preforming eye-tests, visiting to old eye-reports, print and compare them.

Features of Application


Eye-test : Application is desgined to test your both eyes. when you preform a test, very first application tests your left eye. In the second half it tests your right eye. The characters displayed on the screen must be pronounced by you and during the test you must have to recognized the maximum lowest level that you can read for both eye.

Voice recording : During the testing period, users' pronounced characters are recodered by the application it self. therfore, user doesn’t need to memorized them. At the end of the testing they can hear to thier voice by using audio clips generated by the App.

Database : After each test is perfomed, user's eye-reading can be stored in the database. these stored data can be used to further useage. Data of the database only can be accessed for thier original uses.

Filter reports : Data of the database can be filtered according to the enetered data and time. That Feature helps users to find data in more effective way.

Download reports : Data can be donwloaded as two diffrent formats according to thier uses' selection. They can be Download pdf ethier excel sheet.

Print reports : Data in the database can be print as a detail report. This reprot comes with diffrent colums such testID, data & time, leftEye readings and rightEye readings, visons of left eye and vison of right eye.

Test distance of Application


The application provides two types of distances for perform tests. User can select the test distance before they start thier eye-test. Acoordinf to selected distance user should manage the distance between display and thier eyes. usally, distance is provided by feets.

Avalible testing distances are 20fts (6m) and 10fts (3m).
